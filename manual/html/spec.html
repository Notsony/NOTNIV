◆動作環境◆
このゲームを動作させる為には以下の環境が必要となります。
以下の環境以外での動作は保証いたしかねますので、
あらかじめご了承ください。

ＯＳ 日本語版 Windows Vista / 7 / 8
※Windows NT および WindowsRT およびWindowsRT8.1では動作致しません。
CPU 1GHz以上
メモリ 1GB 以上(必須) / 2GB 以上(推奨)
ディスプレイ 1280×720以上の解像度必須
ビデオカード DirectX9.0cに対応したビデオカード（チップ）必須
VRAM 128MB 以上(必須) / 256MB 以上(推奨)
※ご使用のグラフィックボードによっては、満たしていても動作しない可能性があります。
その場合は当マニュアル「お問い合わせ」ページのサポートメールアドレスまでご連絡ください。
HDD インストールするには、約2.9GB空き容量が必要です。
サウンドカード DirectSound に対応したサウンドカード
DirectX DirectX9.0c以降のインストールが必要
  • ※タブレットPCには対応しておりません。
  • ※HDDの必要容量はお使いの環境により多少の上下があります。
  • ※当ゲームはDirectX9.0cを使用しておりますので、お使いのＯＳにDirectX9.0c以上がインストールされている必要があります。
  • ※エミュレータ等の純粋なWindows以外での動作は保証できかねます。
  • ※自作PC/AT機やパーツショップブランド製PC、クロックアップ等の改造を施されたPCでの動作は保証できない場合がございます。
  • ※上記必要環境を満たしている場合でも、お使いの環境により正常に動作しない場合があります。
